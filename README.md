# Sales-Permance-Dashboard-Demo
An interactive Sales Performance &amp; Inventory Analytics Dashboard built with HTML5, CSS3, JS &amp; Chart.js. Track stock levels, revenue, costs, itemized net profits, and margins in real time.


## ✨ Features

- **🚀 Executive KPI Cards**: Real-time tracking of Total Revenue, Cost of Goods Sold (COGS), Gross Profit, Profit Margin %, Inventory Valuation, and Stock Counts.
- **📦 Per-Item Inventory & Profitability Matrix**: Itemized breakdown for every stock item (SKU, Cost, Selling Price, Revenue, Cost, Net Profit, Margin %, and Stock Status).
- **📈 Single-Product & Global Chart Filtering**: Click any item or pick from the header filter to dynamically view single-product financial timelines and charts.
- **📜 Sales Ledger & Record Sales**: Log sales transactions with automatic stock deduction, profit calculation, and payment method tags.
- **📊 Interactive Chart.js Visualizations**: Financial timelines, category breakdown doughnut chart, top profit items bar chart, and stock vs. sales comparisons.
- **📥 CSV Export**: One-click CSV downloads for both inventory matrices and sales logs.
- **🔄 Local Storage & Demo Reset**: Saves data state locally in browser with a 1-click option to reset back to initial realistic dummy dataset.

## 🚀 Quick Start / How to Run

### Option 1: Open Directly in Browser (No Installation Required!)
1. Clone or download this repository.
2. Open `index.html` directly in any web browser (Chrome, Edge, Firefox, Safari).

### Option 2: Run via Local Web Server
If you prefer running via a local server:
```bash
# Using Python
python -m http.server 3000

# OR using Node.js / npx
npx http-server -p 3000
```
Then visit `http://localhost:3000`.
